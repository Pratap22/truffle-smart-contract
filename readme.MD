## Clone the Repo

```bash
git clone git@github.com:Pratap22/truffle-smart-contract.git
```

## Install dependencies

```
yarn install
```

## Setting up the environment

```bash
mv .env.example .env
```

Set values accordingly.

## Running the app

Complie the smart-contract

```bash
truffle compile
```

Migrating the truffle contract

```bash
truffle migrate
```

Testing the test

```bash
truffle test
```
